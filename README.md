This is a project focusing on leveraging on SUI ecosystem to build a dapp that enables enables web3 native and even non web native to be able to use SUI to make everyday transactions like paying bills, subscriptions and many more.
This also enables merchants to be able to create accounts and then create subscriptions or payment systems that will be reflected on every user's dashboard, hence making their services easy to access by anyone anywhere.
The merchants also have access to a dashboard where they can view or monitor how they are doing with analysis tools to help out too.
There would also be integration of AI systems to aid.
